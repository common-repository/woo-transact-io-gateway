WooCommerce payment gateway for transact.io.  Receive payments from $0.01

== Description ==

This is a payment gateway plugin for WooCommerce, that uses transact.io.

Features:

* Payments as low as $0.01
* Transact.io enables publishers to regain control of distribution.
* Transact has lower fees than credit cards or paypal.  For transactions less than $1 the commission is 10%, Over $1 it is 5%.

Read more about us at [transact.io](https://transact.io/)

This plugin is open source under the Apache 2.0 License
The [source code is available](https://gitlab.com/transact/woocommerce-gateway-transact), feel free to create an issue on our git repository

If you need help setting up your account please do not hesitate to contact us:
[https://transact.io/about/contact](https://transact.io/about/contact) or support(at)transact.io

== Installation ==


1. Install or update WooCommerce 3.0+
2. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Woocommerce->Settings->Payments->Transact screen to configure the plugin
4. Sign up with a publisher account on https://transact.io/
5. Go to your Developer Settings->Keys   menu  to find your account ID and secret signing key.  Configure these key on WordPress plugin.
